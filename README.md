# Sokoban Game written in C++11 and SFML

## Authors:
- Giorgio Caculli
- Guillaume Lambert
- Tanguy Taminiau
- Nathan Thaon

## Dependencies
- SFML 2.5.1
- Boost

## Build Tools:
- Cmake
- Autotools

## Building Instructionr:

### Autotools
1. mkdir build
2. cd build
3. autoreconf -fiv ../Sokoban
4. ../Sokoban/configure
5. make
